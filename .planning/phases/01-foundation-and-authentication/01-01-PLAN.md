  <behavior>
    GlassPanel:
    - Renders children inside a div with backdrop-blur, semi-transparent bg, and border classes
    - Supports intensity prop: 'light' (blur-sm, bg-white/[0.03]), 'medium' (blur-md, bg-white/[0.05]), 'heavy' (blur-lg, bg-white/[0.08])
    - Defaults to 'medium' intensity
    - Accepts className prop for customization
    - Accepts 'as' prop for polymorphic element type

    GlassCard (extends GlassPanel):
    - Adds padding (p-6), hover state (hover:bg-white/[0.08]), and transition classes
    - Used for interactive card elements

    Skeleton:
    - Renders shimmer animation div with gradient from-white/[0.06] via-white/[0.12] to-white/[0.06]
    - Supports variant: 'text' (h-4 rounded), 'circular' (rounded-full), 'rectangular' (rounded-lg)
    - Supports width, height, lines props
    - Multi-line mode: last line at 60% width

    useDelayedLoading:
    - When isLoading becomes true, waits delay ms (default 250) before returning true
    - When isLoading becomes false while showing, keeps returning true for at least minDisplay ms (default 500)
    - When isLoading becomes false before delay, never shows skeleton at all
  </behavior>
