- `deleteConfig(id)` -> DELETE `admin/config/${id}`
- `getAdminRatingCriteria()` -> GET `admin/rating-criteria` -> `AdminRatingCriterion[]`
- `createRatingCriterion({ name, description })` -> POST `admin/rating-criteria` -> `AdminRatingCriterion`
- `updateRatingCriterion(id, { name, description })` -> PUT `admin/rating-criteria/${id}` -> `AdminRatingCriterion`
- `toggleRatingCriterion(id)` -> PATCH `admin/rating-criteria/${id}/toggle` -> `AdminRatingCriterion`
- `deleteRatingCriterion(id)` -> DELETE `admin/rating-criteria/${id}`
- `getAdminIngredients({ search? })` -> GET `admin/ingredients` with searchParams -> `Ingredient[]`
- `createIngredient({ name, allergenIds? })` -> POST `admin/ingredients` -> `Ingredient`
- `updateIngredient(id, { name, allergenIds? })` -> PUT `admin/ingredients/${id}` -> `Ingredient`
- `deleteIngredient(id)` -> DELETE `admin/ingredients/${id}`
- `getAdminTags()` -> GET `admin/tags` -> `Tag[]`
- `createTag({ name, parentId? })` -> POST `admin/tags` -> `Tag`
- `updateTag(id, { name?, parentId? })` -> PUT `admin/tags/${id}` -> `Tag`

**AdminTable.tsx** -- Reusable inline-editable table component:
- Generic component: `AdminTable<T>` with props:
  - `columns: Column<T>[]` where `Column` has `{ key: string; header: string; render: (item: T) => ReactNode; editRender?: (value: any, onChange: (v: any) => void) => ReactNode; width?: string }`
  - `data: T[]`, `isLoading: boolean`
  - `searchPlaceholder: string`, `onSearch: (q: string) => void`
  - `onSave: (item: Partial<T>) => Promise<void>`, `onDelete: (item: T) => Promise<void>`, `onCreate: (item: Partial<T>) => Promise<void>`
  - `getId: (item: T) => string | number`
  - `emptyDefaults: Partial<T>` (defaults for new row)
  - `emptyIcon: LucideIcon`, `emptyTitle: string` (for EmptyState)
- State: `editingId` (single row), `editValues` record, `creatingNew` boolean, `searchQuery` string
- Renders inside GlassPanel: search input (uses useDebounce), "+ Add" button, table element with header row + data rows
- Each row: if `editingId === getId(item)`, render editRender cells with Save/Cancel buttons. Otherwise render display cells with Edit/Delete buttons in actions column
- If `creatingNew`, render an editable row at the top of the table body with Save/Cancel
- Delete uses `window.confirm('Delete this item?')`
- Starting edit on one row cancels any active edit (single editingId)
- Loading state: render Skeleton rows (3 rows of skeleton cells)
- Empty state: when data.length === 0 and not loading, render EmptyState with provided icon/title
- Table container: `overflow-x-auto` with `min-w-[600px]` on the table
- Style: glass aesthetic -- white/70 text, white/40 headers, hover:bg-white/5 rows, accent buttons
